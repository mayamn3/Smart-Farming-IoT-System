# Smart Farming IoT System

## Overview

Smart Farming IoT System is an intelligent agricultural monitoring platform designed to collect environmental data and transmit compressed image information over long distances using LoRa communication.

The system combines embedded systems, wireless communication, edge AI and web-based monitoring to help farmers remotely supervise crop conditions and detect potential plant diseases.

## Features

- Environmental monitoring
- Long-range LoRa communication
- ESP32-S3 embedded node
- Raspberry Pi 4 gateway
- Image acquisition and processing
- Convolutional Autoencoder for image compression
- Remote monitoring dashboard
- Real-time sensor visualization

## System Architecture

The system consists of:

1. ESP32-S3 acquisition node
   - Camera module
   - Environmental sensors
   - Data preprocessing

2. AI Compression Layer
   - Convolutional Autoencoder
   - Latent vector generation

3. LoRa Communication
   - Long-range wireless transmission

4. Raspberry Pi Gateway
   - Data reception
   - Image reconstruction
   - Data storage

5. Monitoring Dashboard
   - Sensor visualization
   - Reconstructed image display
